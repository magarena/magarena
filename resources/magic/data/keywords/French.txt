*Absorb N
Si une source infligeait des dégâts à cette créature, prévoyez N de ces dommages.
@Lymph Sliver

*Affinity for [...]
Ce sort vous coûte {1} moins à jeter pour chaque [...] vous contrôlez.
@Frogmite;Razor Golem

*Amplify N
Lorsque cet objet entre dans le champ de bataille, révéler un certain nombre de cartes de votre main qui partagent un type de créature. Ce permanent entre sur le champ de bataille avec N + 1 / + 1 compteurs pour chaque carte révélée de cette façon. Vous ne pouvez pas révéler cette carte ou d'autres cartes qui entrent sur le champ de bataille en même temps que cette carte.
@Kilnmouth Dragon

*Annihilator N
Chaque fois que cette créature attaque, le défenseur du joueur sacrifie N permanents.
@Artisan of Kozilek;Ulamog, the Infinite Gyre;Eldrazi Conscription

*Aura Swap [cost]
[Coût]: Vous pouvez échanger ce permanent avec une carte Aura dans votre main.
@Arcanum Wings

*Awaken N—[cost]
Vous pouvez payer [coût] plutôt que de payer le coût de mana de ce sort en jetant ce sort.
Si le coût de réveil de ce sort a été payé, mettez des compteurs N + 1 / + 1 sur les terres ciblées que vous contrôlez. Cette terre devient une créature élémentaire 0/0 avec hâte. C'est encore une terre.
@Earthen Arms

*Banding
Toutes les créatures avec bandes, et jusqu'à un sans, peuvent attaquer une bande. Les groupes sont bloqués en groupe. Si des créatures avec des bandes que vous contrôlez bloquent ou sont bloquées par une créature, vous divisez les dégâts de combat de cette créature, et non son contrôleur, parmi toutes les créatures qu'elle bloque ou bloque.
@Timber Wolves

*Bands with other [...]
Toutes les créatures nommées [...] peuvent attaquer dans une bande tant qu'une au moins une "bandes avec d'autres [...]" Les groupes sont bloqués en groupe. Si au moins deux créatures nommées [...] vous contrôlez, dont l'une a "des bandes avec d'autres [...]" bloquent ou sont bloquées par la même créature, vous divisez les dégâts de combat de cette créature, pas son contrôleur, Parmi toutes les créatures, elle est bloquée ou bloquante.
@Adventurers' Guildhouse

*Battle Cry
Chaque fois que cette créature attaque, l'autre créature attaquante obtient + 1 / + 0 jusqu'à la fin du tour.
@Accorder Paladin

*Bestow [cost]
Vous pouvez jeter cette carte en payant [coût] plutôt que son coût de mana.
Si vous choisissez de payer le prix de ce sortilège, cela devient un enchantement de l'Aura et gagne une créature enchanteur. Ces effets durent jusqu'à ce que l'une des deux choses se produise: ce sort a une cible illégale au fur et à mesure qu'il résout ou le permanent de ce sortilège devient décisif.
@Nighthowler;Celestial Archon

*Bloodthirst N

@Battering Wurm

*Bloodthirst X
Ce permanent entre dans le champ de bataille avec des compteurs X + 1 / + 1, où X est le dégât total auquel vos adversaires ont été traités ce tour.
@Petrified Wood-Kin

*Bushido N
Chaque fois que cette créature bloque ou devient bloquée, elle obtient + N / + N jusqu'à la fin du tour.
@Hand of Honor

*Buyback [cost]
Vous pouvez payer un supplément [coût] lorsque vous lancez ce sort.
Si le coût du rachat a été payé, placez ce sort dans la main de son propriétaire au lieu du cimetière de ce joueur lorsqu'il résout.
@Lab Rats;Walk the Aeons

*Cascade
Lorsque vous lancez ce sort, exilez les cartes du haut de votre bibliothèque jusqu'à ce que vous exilez une carte non-pays dont le coût de mana converti est inférieur au coût de mana converti de ce sort. Vous pouvez jeter cette carte sans payer son coût de mana. Ensuite, placez toutes les cartes exilées de cette façon qui n'ont pas été jetées au bas de votre bibliothèque dans un ordre aléatoire.
@Captured Sunlight

*Champion an [...]
Quand ce permanent entre dans le champ de bataille, sacrifiez-le à moins d'exiler un autre [...] vous contrôlez.
Lorsque ce permanent quitte le champ de bataille, retournez la carte exilée sur le champ de bataille sous le contrôle de son propriétaire.

*Changeling

*Cipher
Ensuite, vous pouvez exiler cette carte de sortilège encodée sur une créature que vous contrôlez. Chaque fois que cette créature inflige des dégâts de combat à un joueur, son contrôleur peut jeter une copie de cette carte sans payer son coût de mana.
@Shadow Slice

*Conspire
Comme un coût supplémentaire pour jeter ce sort, vous pouvez toucher deux créatures inexploitées que vous contrôlez que chacune partage une couleur avec elle.
Lorsque vous lancez ce sort, si son coût conspiré a été payé, copiez-le. Si le sort a des cibles, vous pouvez choisir de nouvelles cibles pour la copie.

*Convoke

Pour chaque mana générique dans le coût total de ce sort, vous pouvez toucher une créature inexploitée que vous contrôlez plutôt que de payer ce mana.

*Crew N
